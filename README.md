# Frameworkless Server Kata

https://github.com/MYOB-Technology/General_Developer/blob/main/katas/kata-frameworkless-basic-web-application/kata-frameworkless-basic-web-application-enhancements.md


### /

Returns a greeting of all users with the current server time.

```
Hello Cindy and Bob - the time on the server is 3:21 am on Wednesday, 16 September 2020
```
### /users

Returns a list of all users 

```
Cindy 
Bob
```

### /users/name

Returns a single user with corresponding name 

```
/users/cindy => Cindy 
/users/bob => Bob 

```
